## 安装
### Installation with CocoaPods

1.在 `Podfile` 中添加 `pod 'XESTestProject'`。

2.执行 `pod install` 或 `pod update`。

3.导入 `"XESTestProject.h"`。

### 手动安装
1.下载XESTestProject文件夹内的所有内容。

2.将XESTestProject内的源文件添加（拖放）到你的工程。

3.导入XESTestProject.h。
